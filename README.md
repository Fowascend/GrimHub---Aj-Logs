# GrimHub AJ Discord Bot

A Discord bot that sends scheduled brainrot detections to a webhook.

## Commands

- `!status` - Check if bot is running
- `!test` - Send a test detection
- `!next` - Show next scheduled brainrots

## Schedule

| Brainrot | Interval |
|----------|----------|
| Headless Horseman | Every 4 hours |
| Strawberry Elephant | Every 2 hours |
| Meowl | Every 1.5 hours |
| Skibidi Toilet | Every 45 minutes |
| Dragon Cannelloni | Every 15 minutes |

## Deployment

This bot is designed to run on Railway.app

1. Fork this repository
2. Create a new project on Railway.app
3. Add environment variable `DISCORD_TOKEN`
4. Deploy!
